# 자기 평가서

## 1. 가장 잘했다고 생각하는 부분

프로젝트를 하며 패키지의 구조 및 내용을 만들고 깃허브도 vscode에 연결해서 사용하는 방법을 이해해가며 공부하고 개발자로서 필요한 역량과 지식을 더 습득하게 된 점이 잘한 것 같다. 다음에 개발자가 되어 이러한 프로젝트를 하게 되면 이번 경험을 살려서 수월하게 진행할 수 있을 것 같다.

## 2. 아쉬운 부분

처음에는 클래스와 패키지 구조에 대해서도 이해가 부족했고, vscode나 git을 사용하는 것도 미숙해서 ai의 도움을 많이 받은 점이 아쉬웠다.

## 3. 다음에 개선하고 싶은 점

다음에 이런 비슷한 프로젝트를 하게 된다면 이번 프로젝트를 통해 익힌 지식들을 기반으로 처음부터 스스로 클래스를 설계하고 코드를 작성하는 연습을 하고 싶다.

## 4. AI 도구 사용 내역

본 프로젝트에서 Claude(claude.ai)를 활용하였습니다.

- 패키지 디렉터리 구조 설계 및 각 파일 코드 작성 방법 질문
- 클래스 상속(super())과 비공개 메서드(_) 개념 질문
- pytest 테스트 케이스 작성 방법 질문
- PEP 8 오류 수정 방법 질문

강의에서 다루지 않은 기법 중 AI 도움을 받아 사용한 것:
- `import json` 및 `with open()`: JSON 파일 저장/불러오기에 사용.
  강의에서 명시적으로 다루지 않은 표준 라이브러리입니다.
